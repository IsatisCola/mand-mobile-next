# TabBar 标签栏

用于创建不含内容区域的标签栏

# 引入

import { TabBar } from 'mand-mobile-next'

Vue.createApp().component(TabBar.name, TabBar)

immediate初始化后立即就触发一次change事件Booleanfalse
